Os3

A custom-built operating system.

Download this project as a .zip file Download this project as a tar.gz file

Intent

Os3 is an operating system written in x86 assembly code (Intel syntax). It is was started as a project whose main intents are to both educate the author on the workings of and challenges involved in developing the low-level components of an operating system, and explore unorthodox implementations of such systems.

Compatability

As such, systems are being redesigned where possible, so Os3 is not likely to be compatible with any existing software. It uses a custom filesystem, and as such will also not be able to load external files that have not been modified to match its specifications.

Development Builds

No stable release of Os3 is available at the moment, so use the development version with caution. The current version has been tested with, and runs properly on: Bochs (Emulator), and Pentium 4 processors. In order to run Os3, computers will need a minimum of 6.4 megabytes of RAM.
